# LCE Launcher

A simple launcher designed to update and launch the game.

## How it Works

When you open the launcher, it automatically connects to the official repository to check for the latest nightly releases. If a new version is available, it downloads and extracts the files securely to your system (`AppData\Roaming\LCE`).

## Features

* **Auto-Updater:** Ensures you are always running the latest version of the game.
* **Username Configuration:** Easily set and save your in-game username (defaults to Steve).
* **Direct Connection:** Input a specific Server IP and Port right from the launcher.
* **Headless Server Mode:** A built-in toggle to run the application as a dedicated background server.
* **Game Files Access:** A dedicated button to open the game directory instantly.
* **Background Mode:** The launcher hides itself while you are playing and reappears automatically when you close the game.
